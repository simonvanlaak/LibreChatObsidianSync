# Local Test Setup

## Dependencies

Dependencies are installed using `pip install --user` (no virtual environment required).

## Running Tests

### Run all unit tests:
```bash
cd ObsidianSync/McpService
python3 -m pytest tests/unit/ -v
```

### Run specific test file:
```bash
python3 -m pytest tests/unit/test_file_storage.py -v
```

### Run specific test class:
```bash
python3 -m pytest tests/unit/test_file_storage.py::TestFileSearchExclusions -v
```

### Run specific test:
```bash
python3 -m pytest tests/unit/test_file_storage.py::TestFileSearchExclusions::test_search_files_excludes_git_directory -v
```

## Dependencies Installed

All dependencies from `requirements.txt` are installed in `~/.local/lib/python3.10/site-packages/`:
- fastmcp
- uvicorn
- starlette
- aiofiles
- httpx
- gitpython
- pyjwt
- asyncpg
- pgvector
- pytest
- pytest-asyncio
- numpy

## Note

Tests can be run locally via CLI (as per project TDD workflow). Integration tests should still be run in Docker Compose.
